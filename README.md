📝 README – Projetos de Front‑end Interativos
Este repositório contém dois pequenos projetos desenvolvidos com HTML, CSS e JavaScript puro, focados em interfaces interativas, animadas e didáticas para estudos de front‑end.

🧮 Projeto 1 – Calculadora Interativa
Uma calculadora simples e moderna que realiza as quatro operações básicas:

Adição

Subtração

Multiplicação

Divisão

O site apresenta:

Layout responsivo e animado com transições suaves.

Seções de introdução, funcionalidades e área de cálculo.

Resultado em tempo real ao clicar no botão “Calcular”.

🔧 Tecnologias usadas:

HTML5 (estrutura semântica)

CSS3 (estilos, gradientes e animações de entrada)

JavaScript (lógica de cálculo e manipulação do DOM)

🎯 Objetivo:
Demonstrar integração básica entre HTML, CSS e JS, ideal para iniciantes em ADS/Desenvolvimento Web.
